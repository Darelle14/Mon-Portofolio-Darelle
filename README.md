# Portfolio — Darelle Kamche Fotso

## Structure des fichiers

```
portfolio_darelle/
├── index.html       → Page d'accueil
├── apropos.html     → Page À propos (CV, expériences, compétences, certifications)
├── bts.html         → Page BTS SIO (présentation, options SISR/SLAM)
├── e5.html          → Page Épreuve E5 (compétences, documents)
├── e6.html          → Page Épreuve E6 (projets DHCP/DNS et GLPI)
├── contact.html     → Page Contact (formulaire, réseaux sociaux)
├── style.css        → Feuille de style commune à toutes les pages
├── script.js        → JavaScript commun (navigation, upload, formulaire)
└── README.md        → Ce fichier
```

## Comment utiliser

1. Ouvrir `index.html` dans un navigateur web
2. Naviguer entre les pages via le menu en haut
3. Pour ajouter votre CV : aller sur "À propos" > cliquer "Télécharger mon CV"
4. Pour ajouter vos documents E5/E6 : utiliser les boutons "Ajouter le fichier"

## Personnalisation

- Vos informations sont déjà renseignées dans tous les fichiers
- Pour ajouter des certifications : modifier la section correspondante dans `apropos.html`
- Pour modifier les couleurs : changer les variables CSS dans `style.css` (section `:root`)

## Technologies utilisées

- HTML5 / CSS3 / JavaScript Vanilla
- Google Fonts : Playfair Display, DM Sans, DM Mono
- Aucune dépendance externe — fonctionne hors ligne
